# วิธีติดตั้งเพื่อใช้งาน Backend
### 1. ขั้นตอนแรก Dowload Dependencies
```
npm install 
```
### 2. รันคำสั่ง เพื่อเปิดการทำงาน RESTAPI
```

npm run start
```

# การทดสอบแต่เทส api ด้วย postman

```
http://localhost:3000/data
http://localhost:3000/register

```
